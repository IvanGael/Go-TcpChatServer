# Install

**Source**

1. Requirement

```
Enable Telnet on windows
open powershell as administrator
Enable-WindowsOptionalFeature -FeatureName TelnetClient -Online
```

2. Build binary

```
go get;
go build;
```

3. Run the build

```
main
```
